Politique de Confidentialité

Dernière mise à jour : 25 Août 2024

Votre vie privée est importante pour nous. Cette politique de confidentialité explique quelles données personnelles nous collectons auprès de vous et comment nous les utilisons.

1. Données que nous collectons

Nous collectons des données pour fonctionner efficacement et vous fournir les meilleures expériences avec nos services. Vous fournissez certaines de ces données directement, comme lorsque vous créez un compte (nom, email, téléphone, etc.).

2. Comment nous utilisons les données personnelles

Nous utilisons les données que nous collectons pour fournir et améliorer les services que nous offrons. Nous utilisons également les données pour communiquer avec vous, par exemple, en vous informant sur votre compte, les mises à jour de sécurité et les informations sur les produits.

3. Partage de données personnelles

Nous ne partageons pas vos données personnelles avec des tiers sans votre consentement, sauf si la loi l'exige ou pour répondre à une procédure légale.
